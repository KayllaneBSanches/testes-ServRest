Neste repositório estão apresentados os arquivos referentes aos testes que realizei no ServRest utilizando a versão com interface. Aqui encontram-se dois diretórios: 
O primeiro chamado "documentos" reune 3 arquivos .md:
<li><b>plano-de-testes.md:</b> Descrevendo o plano de testes para o sistema em questão.</li>
<li><b>caso-de-testes.md:</b> Relatando cada caso de teste realizado no sistema.</li>
<li><b>relatorio-de-bugs.md:</b> Descrevendo os bugs encontrados no sistema.</li>

O segundo chamado "evidêncis" e nele encontram-se os arquivos comprovando os resultados obtidos nos testes.
